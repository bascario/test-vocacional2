Sistema de Test Vocacional

En la ciudad de Quito, siendo las __________ del día __________ de febrero de 2026, se procede a la formalización de la entrega del proyecto denominado "SISTEMA DE TEST VOCACIONAL", desarrollado bajo el modelo arquitectónico MVC (Model-View-Controller) utilizando tecnologías PHP, JavaScript y bases de datos MySQL.

Quien suscribe, Mg. Rubèn Jaramillo, Docente del Tecnológico Vida Nueva, entrega a la Ing. Cristina López, Directora de Tecnología e Información, el proyecto mencionado con todas sus funcionalidades operativas, código fuente, documentación técnica y base de datos completamente configurada.

El presente proyecto ha sido desarrollado con el propósito de facilitar la evaluación vocacional de los estudiantes del Tecnológico Vida Nueva, mediante un formulario integral que evalúa intereses, habilidades y valores, generando reportes personalizados tanto en formato individual como grupal.

2. Ejecutar: npm start (inicia servidor en http://localhost:8000)
3. Acceder a la URL: http://localhost:8000
